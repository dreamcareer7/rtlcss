RTLCSS
======
Framework for transforming cascading style sheets (CSS) from left-to-right (LTR) to right-to-left (RTL).


## Install
    npm install rtlcss

## Basic usage
```javascript
var rtlcss = require('rtlcss');
var result = rtlcss.process("body { direction:ltr; }");
//result == body { direction:rtl; }
```
